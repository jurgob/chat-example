# chat-example

## install it
```sh

nvm use

npm i

```

## configure it 

add in /etc/hosts the following line:

```sh

127.0.0.1 cane.nip.io

```


## run it

```sh

npm start

```

## test it

```sh 

open http://cane.nip.io:3000/

```

or you can just point your browser at `http://cane.nip.io:3000/` if you are not a bash guru
